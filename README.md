# Milton Cricket Champs (MCC) — Official Website

**Play Hard. Grow Together — Go MCC!!!**

The official website for Milton Cricket Champs, a registered non-profit cricket organization based in the Greater Toronto Area. Founded in 2022, MCC brings together 70+ members across multiple cricket formats and leagues.

🌐 **Live site**: [mcc-miltoncricketchamps.github.io](https://mcc-miltoncricketchamps.github.io)

---

## What's on the Site

| Page | Description |
|---|---|
| **Home** | Hero banner, club stats, latest achievements, photo highlights |
| **About** | Our story, mission, values, and the formats we play (hard tennis, tape ball, leather) |
| **Team** | Leadership profiles and team roster |
| **Achievements** | Full timeline of tournament results and championships |
| **Gallery** | Photo gallery from matches, events, and team moments |
| **Videos** | YouTube match highlights and event recaps |
| **Analytics** | Live player stats, match history, and win rates — powered by CricHeroes data |
| **Sponsorship** | Sponsorship tiers, benefits, and downloadable deck |
| **Contact** | Get in touch, join the club, or follow us on social media |

## Tech Stack

- **Static HTML/CSS/JS** — no framework, fast and lightweight
- **GitHub Pages** — hosted directly from this repo
- **CricHeroes integration** — player and match stats scraped and refreshed automatically
- **YouTube sync** — new match videos pulled in automatically
- **Auto-refresh** — site data updates twice daily (10 AM & 4 PM ET)

## How It Works

The site is generated by a Python build script that pulls in live data from CricHeroes and YouTube, then outputs static HTML pages. A scheduled task runs the full pipeline (scrape → sync → build → push) twice a day to keep everything current.
